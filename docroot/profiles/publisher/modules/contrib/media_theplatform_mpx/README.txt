Media: thePlatform mpx
======================

The Media: thePlatform mpx module integrates with the Media module 
to make your mpxPlayers and mpxMedia on thePlatform.com available in 
Drupal as File Entities.


-- REQUIREMENTS --

* An administrator account on thePlatform mpx
* Media 7.x-2.x (http://drupal.org/project/media)
* File Entity (http://drupal.org/project/file_entity)


-- CONFIGURATION --

See http://drupal.org/project/media_theplatform_mpx
